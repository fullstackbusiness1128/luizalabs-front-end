- Favorite Places
An app to search and save your favorite places by CEP

Features
- Search place by CEP
- Be able to save in favorite places

Flow
- Search by CEP
- Show place info
- Clicking on Favorite button triggers a modal with
  options to add a name and description
- Able to navigate through favorites or search more
